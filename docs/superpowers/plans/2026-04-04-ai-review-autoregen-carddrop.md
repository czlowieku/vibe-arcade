# AI Game Review, Auto-Regen & Card Drop Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add AI-powered game reviews from NPCs, auto-regenerate broken games, show actionable suggestions, and let players drag cards onto machines to enhance existing games.

**Architecture:** Server gets a new `/api/review` endpoint (uses Haiku for cheap reviews). Client tracks crash counts per machine, calls review after NPC finishes, blends AI rating with score-based rating. Broken games (2+ crashes/broken flags) auto-regenerate from saved recipe. Suggestions accumulate on machines and can be applied. Card drag-and-drop from card bar onto machines triggers regeneration with added card.

**Tech Stack:** Express, Anthropic SDK (claude-haiku-4-5-20251001), Three.js, vanilla JS

```js
import Anthropic from '@anthropic-ai/sdk';

export async function reviewGame(apiKey, gameCode, genre, theme, modifier, npcScore) {
  const client = new Anthropic({ apiKey });

  const prompt = `You are a game critic reviewing a mini-game in an arcade. The game was generated by AI.

Game genre: ${genre}
Theme: ${theme}
${modifier ? `Modifier: ${modifier}` : ''}
NPC player scored: ${npcScore} points

Here is the game's source code:
\`\`\`javascript
${gameCode.slice(0, 4000)}
\`\`\`

Review this game. Respond in JSON only:
{
  "rating": <1-5 integer, how fun/polished is this game>,
  "isBroken": <true if game crashes, freezes, or is unplayable>,
  "feedback": "<1 sentence summary of the game quality>",
  "suggestions": ["<specific actionable improvement 1>", "<specific actionable improvement 2>"]
}

Be honest. If the game is broken or unplayable, say so. If it works but isn't fun, rate it low. Max 2 suggestions.`;

  const response = await client.messages.create({
    model: 'claude-haiku-4-5-20251001',
    max_tokens: 300,
    messages: [{ role: 'user', content: prompt }],
  });

  const text = response.content[0].text;
  try {
    // Extract JSON from response (may have markdown fences)
    const jsonMatch = text.match(/\{[\s\S]*\}/);
    if (!jsonMatch) throw new Error('No JSON in response');
    return JSON.parse(jsonMatch[0]);
  } catch (e) {
    console.error('Failed to parse review:', text);
    return { rating: 3, isBroken: false, feedback: 'Review failed to parse', suggestions: [] };
  }
}
```

```js
    const scoreRating = hadRunner
      ? this._scoreToRating(gameScore, npc.personality.standards)
      : this.reputation.calculateRating(npc.personality.standards, 3);

    // Check for crashes

    // Fire-and-forget AI review (don't block NPC flow)
    this._callReview(machine, gameScore).then(review => {

      const aiRating = review.rating;
      const blended = Math.round(0.4 * scoreRating + 0.6 * aiRating);

      // Update reputation with blended rating
      this.reputation.addRating(machine.index, finalRating);

      if (review.suggestions && review.suggestions.length > 0) {

      if (review.isBroken || crashCount > 0) {
```

```js
  _autoRegenerate(machine, recipe, feedback) {
    const apiKey = getApiKey();
    if (!apiKey) return;

    const extraInstructions = `CRITICAL: The previous version of this game was broken/unplayable. Issue: ${feedback || 'game crashes or freezes'}. Generate a WORKING version. Test your logic carefully.`;

    machine.state = 'generating';
    machine.streamedCode = '';

    fetch('/api/generate', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        genre: recipe.genre,
        theme: recipe.theme,
        modifier: recipe.modifier,
        cardLevels: recipe.cardLevels || { genre: 1, theme: 1, modifier: 0 },
        extraInstructions,
        apiKey,
      }),
    }).then(async response => {
      const reader = response.body.getReader();
      const decoder = new TextDecoder();
      let buffer = '';
      while (true) {
        const { done, value } = await reader.read();
        if (done) break;
        buffer += decoder.decode(value, { stream: true });
        const lines = buffer.split('\n');
        buffer = lines.pop();
        for (const line of lines) {
          if (!line.startsWith('data: ')) continue;
          const data = JSON.parse(line.slice(6));
          if (data.type === 'done') {
            machine.setGame(data.gameCode, data.title, data.description);
            const saved = this.gameState.machines[machine.index];
            if (saved) {
              saved.gameCode = data.gameCode;
              saved.title = data.title;
              saved.description = data.description;
              saved.brokenCount = 0;
              saved.suggestions = [];
            }
            machine.suggestions = [];
            machine.brokenCount = 0;
            this.save();
          }
        }
      }
    }).catch(err => {
      console.error('Auto-regen failed:', err);
      machine.state = 'ready';
      machine.drawReady();
    });
  }
```
